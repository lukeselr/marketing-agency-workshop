# While You Wait — engagement loop during platform approvals

LinkedIn Marketing API takes 1-5 business days. Google Ads developer token (Basic) is auto-approved within 24h. Don't sit idle — these tasks land before approvals do, and they directly improve campaign quality.

## Tier 1 — 60 minutes today

1. **Draft 3 organic LinkedIn posts** in your voice
   - Use the `linkedin-automation` skill or `social-content` skill
   - Voice fingerprint already extracted in Phase 1 — these posts come out on-brand by default
   - Schedule for Mon / Wed / Fri this week

2. **Draft 6 ad concept variations**
   - Use the `ad-creative` skill with your `voice-fingerprint.json`
   - 3 hooks × 2 angles = 6 variants
   - Save to `.state/creatives/while-you-wait/`

## Tier 2 — 30 minutes tomorrow

3. **Draft Lead Gen Form copy**
   - Headline + 3-step form + privacy line
   - Use `copywriting` skill

4. **Install Pixel + Insight Tag now** (works pre-API approval)
   - Run Phase 6 — tags fire on user visits regardless of API status
   - Builds retargeting audience while you wait

## Tier 3 — when boredom strikes

5. **Re-read the SKILL.md + the relevant Phase doc** to align mental model
   - `~/.claude/skills/marketing-agency/SKILL.md`
   - `~/.claude/skills/marketing-agency/PHASES/<current-phase>.md`

6. **Check competitor changes**
   - Re-run `competitor-scraper` agent against top 3 competitors
   - Note any new ads they launched while you were waiting
