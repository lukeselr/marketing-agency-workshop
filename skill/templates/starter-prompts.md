# Starter prompts for new users

Paste any of these into Claude Code after `marketing-agency` is installed.

## "Run the full onboarding"
> Run the marketing-agency skill against my business at workshop.selrai.com.au. Pause for me to confirm at each phase.

## "Just discover, don't onboard yet"
> Use the marketing-agency Phase 1 to scrape my business and competitors. Stop after the baseline doc — I'll review.

## "Generate creative for an existing campaign"
> Use the marketing-agency creative-generator agent. My business baseline already exists. Make 30 ad variants for the Sydney Workshop concept across Meta + LinkedIn + Google.

## "Set up tracking only"
> Run Phase 6 of marketing-agency. Site host detection + Pixel + Insight Tag + Google Ads conversion install. Do not touch ad accounts.

## "Kill-rule monitor only"
> Deploy the marketing-agency kill-rule monitor as a launchd job on my Mac. Daily 9am AEST. Telegram alerts via my existing bot.
