# Phase 2 — Strategy Selection

Reads `business-baseline.md` + ads-strategy memory + `connector-recommender` skill output.

## Output: 3 ranked options

Each option includes:
- Recommended platforms (Meta / LinkedIn / Google)
- Spend split (e.g. 50/30/20)
- Lead generation mechanism (Lead Gen Form / Landing Page / Phone)
- Estimated CPA + scaling cap
- Risk factors

User picks one. Selection drives Phase 3 platform onboarding.
