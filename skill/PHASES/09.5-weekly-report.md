# Phase 9.5 — Weekly client report

Auto-generated every Monday 7am AEST. The owner gets the same one-page report a real agency would email.

## How it runs

```bash
bash ~/.claude/skills/marketing-agency/scripts/weekly-report.py
```

## What it produces

- `~/marketing/reports/YYYY-WW.html` (printable, embeddable)
- `~/marketing/reports/YYYY-WW.json` (raw data for downstream)
- `~/marketing/this-week.md` (refreshed)

## Sections

- **Top-line**: spend, leads, CPA, ROAS vs goal, ▲/▼ vs last week
- **Per-platform**: Meta / LinkedIn / Google Ads with connection status
- **Winners**: 3 best ads/campaigns with stat-sig delta (filled by `delegate-analytics-dashboard`)
- **Killed**: anything paused by `kill-rule-monitor.py`
- **Pipeline**: GHL CRM new leads / MQL / SQL counts (via `delegate-analytics-dashboard`)
- **Creative fatigue**: CTR decline curves
- **Next 3 actions**: from `claude-ads/skills/ads-plan` template

## Cron deployment

Auto-installed by Phase 9 deployer. Choose one:
- AWS server cron: `0 7 * * 1 bash ~/.claude/skills/marketing-agency/scripts/weekly-report.py`
- macOS launchd: same command, weekly trigger
- Anthropic Routine: cloud cron (Mondays 7am AEST = Sundays 9pm UTC)
